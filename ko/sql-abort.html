ABORT

ABORT - 현재 트랜잭션을 중단합니다.

시놉시스

ABORT [ 작업 | 트랜잭션 ] [ 및 [ 아니오 ] 체인 ]을 중단합니다.

설명

ABORT는 현재 트랜잭션을 롤백하고 트랜잭션에서 수행한 모든 업데이트를 삭제합니다. 이 명령은 표준 SQL 명령인 ROLLBACK과 동작이 동일하며 기록상의 이유로만 존재합니다.

매개변수

work
트랜잭션

선택적 키워드. 아무 효과가 없습니다.

AND CHAIN

AND CHAIN을 지정하면 방금 완료된 트랜잭션과 동일한 트랜잭션 특성( SET TRANSACTION 참조)을 가진 새 트랜잭션이 즉시 시작됩니다. 그렇지 않으면 새 트랜잭션이 시작되지 않습니다.

참고

트랜잭션을 성공적으로 종료하려면 COMMIT을 사용합니다.

트랜잭션 블록 외부에서 ABORT를 실행하면 경고가 표시되며, 그렇지 않으면 아무런 효과가 없습니다.

예시

모든 변경 내용을 취소하려면

ABORT;

호환성

이 명령은 과거에 존재했던 PostgreSQL 확장 명령입니다. ROLLBACK은 이에 상응하는 표준 SQL 명령입니다.
