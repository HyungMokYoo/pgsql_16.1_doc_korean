ALTER EVENT TRIGGER

ALTER EVENT TRIGGER - 이벤트 트리거의 정의를 변경합니다.

Synopsis

ALTER EVENT TRIGGER name DISABLE ALTER EVENT TRIGGER name ENABLE [ REPLICA | ALWAYS ] ALTER EVENT TRIGGER name OWNER TO { new_owner | CURRENT_ROLE | CURRENT_USER | SESSION_USER } ALTER EVENT TRIGGER name RENAME TO new_name

설명

ALTER EVENT TRIGGER는 기존 이벤트 트리거의 속성을 변경합니다.

이벤트 트리거를 변경하려면 수퍼유저 권한이 있어야 합니다.

파라미터

name

변경할 기존 트리거의 이름입니다.

new_owner

이벤트 트리거의 새 소유자의 사용자 이름입니다.

new_name

이벤트 트리거의 새 이름입니다.

비활성화/활성화[ 복제본 | 항상 ]를 선택합니다.

이 양식은 이벤트 트리거의 실행을 구성합니다. 비활성화된 트리거는 여전히 시스템에 알려지지만 트리거 이벤트가 발생할 때 실행되지 않습니다. 세션_복제_역할도 참조하세요.

호환성

SQL 표준에는 ALTER EVENT TRIGGER 문이 없습니다.
