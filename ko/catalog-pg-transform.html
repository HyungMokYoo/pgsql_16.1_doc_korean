53.57. pg_transform #

pg_transform 카탈로그는 데이터 유형을 절차적 언어에 맞게 조정하는 메커니즘인 변환에 대한 정보를 저장합니다. 자세한 내용은 변환 생성을 참고한다.

표 53.57. pg_transform 열

열 유형

설명

OID OID

행 식별자

trftype oid( pg_type.oid 참조)

이 변환의 대상 데이터 타입의 OID

trflang oid( pg_language.oid 참조)

이 변환이 적용되는 언어의 OID

trffromsql regproc(참조 pg_proc.oid)

입력 데이터 유형을 절차적 언어(예: 함수 매개변수)로 변환할 때 사용할 함수의 OID입니다. 기본 동작을 사용해야 하는 경우 0이 저장됩니다.

trftosql regproc(참조 pg_proc.oid)

절차적 언어의 출력(예: 반환 값)을 데이터 유형으로 변환할 때 사용할 함수의 OID입니다. 기본 동작을 사용해야 하는 경우 0이 저장됩니다.
