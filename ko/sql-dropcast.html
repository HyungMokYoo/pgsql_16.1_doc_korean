DROP CAST

DROP CAST - 캐스트 제거

개요

DROP CAST [ IF EXISTS ](source_type AS target_type) [ CASCADE | RESTRICT ]

설명

DROP CAST는 이전에 정의된 캐스트를 제거합니다.

형 변환을 삭제하려면 소스 또는 대상 데이터 유형에 대한 소유 권한이 있어야 합니다. 이는 형 변환을 만드는 데 필요한 권한과 동일합니다.

매개변수

IF EXISTS

형 변환이 존재하지 않는 경우 오류를 발생시키지 않습니다. 이 경우 알림이 발행됩니다.

source_type

형 변환의 소스 데이터 유형의 이름입니다.

대상_유형

형변환의 대상 데이터 유형의 이름입니다.

캐스케이드
제한

이 키워드는 형변환에 대한 종속성이 없으므로 아무런 영향을 미치지 않습니다.

예제

텍스트 유형에서 int 유형으로 형변환을 삭제하려면 다음과 같이 하세요:

DROP CAST (text AS int);

호환성

DROP CAST 명령은 SQL 표준을 준수합니다.
