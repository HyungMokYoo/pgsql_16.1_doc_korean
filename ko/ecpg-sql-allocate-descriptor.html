ALLOCATE DESCRIPTOR

ALLOCATE DESCRIPTOR - SQL 설명자 영역을 할당합니다.

개요

설명자 이름할당

설명

ALLOCATE DESCRIPTOR는 PostgreSQL 서버와 호스트 프로그램 간에 데이터를 교환하는 데 사용할 수 있는 새 명명된 SQL 디스크립터 영역을 할당합니다.

설명자 영역은 사용 후 DEALLOCATE DESCRIPTOR 명령을 사용하여 해제해야 합니다.

파라미터

이름 #

대소문자를 구분하는 SQL 설명자의 이름입니다. SQL 식별자 또는 호스트 변수일 수 있습니다.

예제

EXEC SQL ALLOCATE DESCRIPTOR mydesc;

호환성

ALLOCATE DESCRIPTOR는 SQL 표준에 지정되어 있습니다.
