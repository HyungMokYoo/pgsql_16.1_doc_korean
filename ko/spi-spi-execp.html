SPI_execp

SPI_execp - 읽기/쓰기 모드에서 문을 실행합니다.

시놉시스

int SPI_execp(SPIPlanPtr plan, Datum * values, const char * nulls, long count)

설명

SPI_execp는 SPI_execute_plan과 동일하지만, 후자의 읽기 전용 파라미터는 항상 거짓으로 간주됩니다.

인수

SPIPlanPtr 계획

준비된 문( SPI_prepare에서 반환)

데이텀 * 값

실제 매개변수 값의 배열입니다. 문장의 인수 수와 길이가 같아야 합니다.

const char * 널

어떤 매개변수가 널인지 설명하는 배열입니다. 문장의 인수 수와 길이가 같아야 합니다.

nulls가 NULL이면 SPI_execp는 매개변수가 null인 것으로 가정합니다. 그렇지 않으면, nulls배열의 각 항목은 해당 매개변수 값이 널이 아닌 경우 ' ', 해당 매개변수 값이 널인 경우 'n' 이어야 합니다. (후자의 경우 해당 값 항목의 실제 값은 중요하지 않습니다.) null은 텍스트 문자열이 아니라 배열이므로 '\0' 종결자가 필요하지 않습니다.

long count

반환할 최대 행 수 또는 제한이 없는 경우 0입니다.

반환 값

SPI_execute_plan을 참조하십시오.

성공하면 SPI_processed 및 SPI_tuptable은 SPI_execute와 같이 설정됩니다.
