47.1. 인터페이스 함수 번호

SPI_connect - C 함수를 SPI 매니저에 연결합니다.
SPI_finish - SPI 매니저에서 C 함수 연결 끊기
SPI_execute - 명령 실행
SPI_exec - 읽기/쓰기 명령 실행
SPI_execute_extended - 줄을 벗어난 매개 변수로 명령 실행
SPI_execute_with_args - 줄을 벗어난 매개 변수를 사용하여 명령 실행
SPI_prepare - 아직 실행하지 않고 문을 준비합니다.
SPI_prepare_cursor - 아직 실행하지 않고 문을 준비합니다.
SPI_prepare_extended - 아직 실행하지 않고 문을 준비합니다.
SPI_prepare_params - 아직 실행하지 않고 문을 준비합니다.
SPI_getargcount - SPI_prepare에 의해 준비된 문에 필요한 인자의 수를 반환합니다.
SPI_getargtypeid - SPI_prepare에 의해 준비된 문의 인자에 대한 데이터 타입 OID를 반환합니다.
SPI_is_cursor_plan - SPI_prepare에 의해 준비된 문이 SPI_cursor_open과 함께 사용될 수 있는 경우 true를 반환합니다.
SPI_execute_plan - SPI_prepare에 의해 준비된 문을 실행합니다.
SPI_execute_plan_extended - SPI_prepare에 의해 준비된 문을 실행합니다.
SPI_execute_plan_with_paramlist - SPI_prepare에 의해 준비된 문을 실행합니다.
SPI_execp - 읽기/쓰기 모드에서 문을 실행합니다.
SPI_cursor_open - SPI_prepare로 생성된 문을 사용하여 커서를 설정합니다.
SPI_cursor_open_with_args - 쿼리와 매개 변수를 사용하여 커서를 설정합니다.
SPI_cursor_open_with_paramlist - 매개 변수를 사용하여 커서를 설정합니다.
SPI_cursor_parse_open - 쿼리 문자열과 파라미터를 사용하여 커서를 설정합니다.
SPI_cursor_find - 이름으로 기존 커서 찾기
SPI_cursor_fetch - 커서에서 일부 행을 가져옵니다.
SPI_cursor_move - 커서 이동
SPI_scroll_cursor_fetch - 커서에서 일부 행을 가져옵니다.
SPI_scroll_cursor_move - 커서 이동
SPI_cursor_close - 커서 닫기
SPI_keepplan - 준비된 문 저장
SPI_saveplan - 준비된 문 저장
SPI_register_relation - 임시 명명된 관계를 SPI 쿼리에서 이름으로 사용 가능하게 만듭니다.
SPI_unregister_relation - 레지스트리에서 임시 명명된 관계 제거
SPI_register_trigger_data - SPI 쿼리에서 임시 트리거 데이터를 사용할 수 있게 합니다.
