PostgreSQL에는 네 가지 기본 문서 형식이 있습니다:

  • 일반 텍스트, 설치 전 정보용

  • HTML, 온라인 검색 및 참조용

  • PDF, 인쇄용

  • 빠른 참조를 위한 매뉴얼 페이지.

또한 다양한 구현 문제를 문서화한 일반 텍스트 README 파일은 PostgreSQL 소스 트리 전체에서 찾을 수 있습니다.

HTML 문서와 매뉴얼 페이지는 표준 배포의 일부이며 기본적으로 설치됩니다. PDF 형식 문서는 별도로 다운로드할 수 있습니다.
